# dev-lab

Repositório de experimentos e aprendizado de **JavaScript**.  
Cada pasta contém um mini-projeto para praticar conceitos básicos e intermediários de JS.

## Experimentos

1. **Calculadora Simples**  
   Uma calculadora básica para entender eventos, funções e manipulação de inputs.

2. **To-Do List**  
   Lista de tarefas dinâmica que ensina arrays, loops e manipulação do DOM.

3. **Gerador de Cores Aleatórias**  
   Gerador de cores com JS para praticar Math.random e alteração de estilos.

4. **Mini Quiz**  
   Pequeno quiz interativo com perguntas e respostas para treinar condicionais e loops.
