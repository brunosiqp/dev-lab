# RPG Battle Simulator - Special

A fun and interactive RPG battle simulator featuring unique characters like Guri, Shawty, Fei, and Birrll.  
Players can choose characters and watch turn-based battles with normal attacks and special abilities.  
This project is ideal for learning JavaScript, JSON, and DOM manipulation.  

---

## Features / Funcionalidades / Fonctionnalités

- Turn-based battles between two characters  
- Unique characters with special skills (damage, heal, defense boost)  
- Random critical hits for normal attacks  
- Battle log showing each action and updated HP  
- Uses JSON to store character data  
- Interactive dropdown to select characters  

---

## How to Use / Como Usar / Comment Utiliser

1. Clone the repository / Clone o repositório / Clonez le dépôt:
   ```bash
   git clone https://github.com/seu-usuario/dev-lab.git
   ```
2. Open the project in your browser / Abra o projeto no navegador / Ouvrez le projet dans le navigateur:  
   `projects/rpg-simulator/index.html`
3. Select two different characters and click **Start Battle** / Selecione dois personagens diferentes e clique em **Start Battle** / Sélectionnez deux personnages différents et cliquez sur **Start Battle**
4. Watch the battle unfold in the battle log / Veja a batalha se desenrolar no log / Regardez le combat dans le journal de bataille

## Characters / Personagens / Personnages

- **Guri** – Balanced stats with Power Strike and Heal  
- **Shawty** – High attack, Quick Slash and Dodge abilities  
- **Fei** – High attack, Fury ability  
- **Birrll** – High defense, Shield Bash and Reinforce abilities
