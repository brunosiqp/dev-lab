# Mini Task Tracker / Rastreador de Tarefas / Suivi de Tâches

A small sandbox project to **learn, experiment, and test** Python, Java, HTML, and CSS integration.

Um pequeno projeto sandbox para **aprender, experimentar e testar** integração de Python, Java, HTML e CSS.

Petit projet sandbox pour **apprendre, expérimenter et tester** l'intégration de Python, Java, HTML et CSS.

---

## Usage / Como Usar / Utilisation

**Frontend:** Open `frontend/index.html` in a browser.  
**Backend Java:** Compile and run `TaskAPI.java`. Run tests with JUnit.  
**Python Client:** Run `task_client.py` to simulate backend interaction.  

**Frontend:** Abra `frontend/index.html` no navegador.  
**Backend Java:** Compile e execute `TaskAPI.java`. Execute os testes com JUnit.  
**Cliente Python:** Execute `task_client.py` para simular interação com o backend.

---

## Purpose / Objetivo / Objectif

- Learn basic full-stack integration  
- Test Java, Python, HTML, and CSS together  
- Experiment with functional tests in Java and Python  

- Aprender integração básica full-stack  
- Testar Java, Python, HTML e CSS juntos  
- Experimentar testes funcionais em Java e Python  

- Apprendre l'intégration full-stack de base  
- Tester Java, Python, HTML et CSS ensemble  
- Expérimenter des tests fonctionnels en Java et Python
