# GitHub Language Usage Index

This project analyzes the most used programming languages on GitHub using the GitHub REST API.

## Features

- Fetches repositories using GitHub Search API
- Aggregates language usage
- Generates ranking table
- Creates bar chart of top languages
- Supports GitHub token for higher rate limits

## Requirements

- Python 3.10+
- GitHub personal access token (recommended)
